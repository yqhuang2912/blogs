# 科学空间博客 - 前端设计复刻

这是一个复刻自 [科学空间 (kexue.fm)](https://kexue.fm/) 的前端设计项目，用于学习优秀的博客网站设计。

## 项目简介

**科学空间**是一个专注于数学、机器学习、优化算法等学术内容的技术博客，由苏剑林创建。本项目复刻了其简洁优雅的设计风格。

## 设计特点

### 1. 布局设计
- **固定侧边栏**: 左侧固定导航栏，深色主题
- **主内容区**: 右侧文章列表，白色背景
- **响应式设计**: 完美适配桌面端和移动端

### 2. 视觉风格
- **极简主义**: 简洁的排版，突出内容
- **学术风格**: 适合技术博客的阅读体验
- **配色方案**: 
  - 侧边栏: `#2c3e50` (深蓝灰)
  - 主题色: `#3498db` (亮蓝色)
  - 文字: `#333` (深灰)

### 3. 交互效果
- 卡片悬停效果
- 平滑滚动
- 渐进式加载动画
- 返回顶部按钮

## 使用方法

### 直接打开
双击 `index.html` 文件即可在浏览器中查看。

### 使用本地服务器（推荐）
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server
```

然后在浏览器访问: `http://localhost:8000`

## 技术栈

- **HTML5**: 语义化标签
- **CSS3**: 
  - Flexbox 布局
  - CSS Grid
  - CSS 动画和过渡
  - 媒体查询（响应式设计）
- **JavaScript (ES6+)**:
  - DOM 操作
  - 事件监听
  - Intersection Observer API
  - 平滑滚动

## 响应式断点

- **桌面端**: > 1024px
- **平板端**: 768px - 1024px
- **移动端**: < 768px

## 性能优化建议

1. **图片优化**: 使用 WebP 格式，添加懒加载
2. **CSS 优化**: 使用 CSS 压缩工具
3. **JavaScript 优化**: 代码分割，按需加载
4. **缓存策略**: 设置合理的缓存头
5. **CDN**: 使用 CDN 加速静态资源

## 自定义配置

### 修改配色
在 `styles.css` 中修改以下变量：

```css
/* 主题色 */
.sidebar {
    background-color: #2c3e50; /* 侧边栏背景 */
}

.post-date {
    background-color: #3498db; /* 日期标签 */
}
```

### 添加新文章
在 `index.html` 中复制 `.post-card` 结构并修改内容。
